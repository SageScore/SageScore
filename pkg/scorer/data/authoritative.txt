# Authoritative domains list — SageScore v0.1
#
# Used by the citation-worthiness analyser (docs/methodology.md §6) to
# detect outbound links to high-authority sources. One bare domain per
# line, lowercase, no scheme, no path. Lines starting with '#' are
# comments. Subdomains are NOT matched recursively — list each one
# explicitly.
#
# Curation principles:
#   - Globally recognised reference, government, education, or major
#     news outlets.
#   - No commercial review/affiliate sites.
#   - No paywalled-by-default sources unless the title is widely
#     considered authoritative (e.g. NYT, FT).
#   - Prefer top-level institutional domains (nih.gov) over individual
#     sub-projects.
#
# Updates ship as PRs and trigger a scorer-version patch bump.
